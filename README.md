# Готовый шаблон модуля для Opencart 3

### Как пользоваться?


##### Копируем данные к себе на сервер
```
git clone https://github.com/manwhocode/Mod_3.git
```

##### Переходим в папку с файлами
```
cd путь/module
```

##### Запускаем шелл - в конце добавляем имя модуля
```
./clone.sh имя_модуля
```
